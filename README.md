# ShipShapes.jl

Parametric and tabulated ship hull signed distance functions for
[WaterLily.jl](https://github.com/WaterLily-jl/WaterLily.jl). Private,
WIP. See [PLAN.md](./PLAN.md) for the full design.

Wigley (analytic), KCS (KRISO container ship), DTC (Duisburg test case),
and DTMB 5415 (US Navy combatant) as `AbstractBody` SDFs.
